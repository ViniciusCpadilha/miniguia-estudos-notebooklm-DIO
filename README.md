# Inteligência Artificial & Machine Learning

Objetivos da criação no NotebookLM:
- Compreender o que é Inteligência Artificial e como ela se diferencia do Machine Learning
- Conhecer os principais tipos de aprendizado de máquina (supervisionado, não supervisionado e por reforço)
- Entender como os algoritmos "aprendem" a partir de dados
- Identificar aplicações reais de IA e ML no dia a dia
- Criar uma base sólida de conhecimento para evoluir para temas mais avançados como Deep Learning e Redes Neurais
